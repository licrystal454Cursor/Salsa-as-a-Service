<Update label="March 2024" description="v2.5.0" tags={["Release", "API", "AI", "IoT", "Performance", "Fixes"]} rss="v2.5.0 introduced GraphQL, enhanced AI forecasting, IoT Sensor Hub v2, major performance upgrades, and important bug fixes.">
  ## Major New Features

  ### GraphQL API Launch
  - **GraphQL Endpoint**: New flexible query language for advanced integrations
  - **Real-Time Subscriptions**: WebSocket-based subscriptions for live data updates
  - **Schema Introspection**: Self-documenting API with built-in schema explorer
  - **Query Optimization**: Automatic query optimization and batching

  ### Enhanced AI Forecasting v2
  - **Multi-Model Ensemble**: Combines 5 ML models for 98% accuracy
  - **Contextual Forecasting**: Incorporates local events, weather, social trends
  - **Confidence Intervals**: Statistical confidence ranges for all predictions
  - **What-If Analysis**: Simulate different scenarios and their impact on demand

  ### IoT Sensor Hub v2
  - **Edge Computing**: On-device processing for faster response times
  - **Mesh Networking**: Sensors communicate directly for improved reliability
  - **Predictive Maintenance**: AI predicts equipment failures 14 days in advance
  - **Visual Inspection AI**: Computer vision for automated quality inspection

  ## Major Improvements

  - **Performance**: 65% faster API response times with new caching layer
  - **Mobile App**: Complete redesign with offline-first architecture
  - **Security**: Added FIDO2/WebAuthn support for passwordless authentication
  - **Integrations**: Added 15 new integration partners (total now 65+)
  - **Dashboard**: Redesigned dashboard with customizable widgets and layouts

  ## Bug Fixes

  - Fixed rare race condition in multi-location inventory sync
  - Resolved GraphQL subscription memory leak
  - Corrected timezone handling in forecast calculations
  - Fixed CSV export formatting for international number formats
</Update>
